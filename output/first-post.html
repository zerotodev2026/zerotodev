嗨，我是 ZeroToDev。

起因

最近 AI 很火，我装了个 Claude 试了试，没想到一用就上瘾了。

从一个完全不懂代码的小白，一路折腾到现在，我已经搞了不少东西：

  • Claude Code — AI 编程助手，装了
  • VS Code — 代码编辑器，装了
  • OpenClaw — AI 助手平台，也装了
  • DeepSeek API — 买了
  • 小米 API — 也买了
  • 还用 Python 做了第一个程序 CleanBuddy，已经打包好了

前前后后花了几十块钱，但比起学到的东西，太值了。

为什么要做这个网站

做出来的东西总得有个地方放吧？而且我也想记录下整个过程——从啥都不会，到能做出东西来。

这个网站就是我的实验基地：

  • 放我做的软件和小工具
  • 记录学习过程中的坑和收获
  • 看看一年后我能走到哪

接下来的计划

  • 把 CleanBuddy 完善一下
  • 做一个小游戏
  • 继续学 Python 和 Web 开发

欢迎围观一个编程小白的逆袭之路 🚀
