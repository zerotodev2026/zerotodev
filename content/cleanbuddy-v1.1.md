Title: CleanBuddy v1.1 — UI大改 + 多项修复
Date: 2026-06-05
Category: Project
Tags: cleanbuddy, python, qt, release

今天对 CleanBuddy 做了一次大更新，从 UI 到功能都做了不少改进和修复。

<!-- more -->

## 🎨 UI 大改

**无边框窗口。** 去掉了 Windows 默认标题栏，用自定义的 — □ ✕ 按钮放在右上角。窗口四边可以拖拽调整大小，顶部可以拖动和双击最大化，和系统原生窗口一样好用。

**左右分栏布局。** 扫描结果和迁移页都改成左右分栏了。以前分析和说明都挤在底部一小块地方，现在放在右侧，内容再多也不怕。

**暗色弹窗统一。** 之前迁移和清理的提示弹窗都是 Windows 默认的白底，和暗色主题格格不入。现在全部改成深色自定义弹窗，文字还能选中复制。

## 🐛 Bug 修复

**临时文件重复扫描。** Windows 上 `%TEMP%`、`%TMP%`、`AppData\Local\Temp` 指向同一个目录，之前会被扫出 3 条一模一样的项。用路径归一化去重后只显示 1 条。

**迁移页分类丢失。** 扫描结果导入迁移页时所有项都被硬编码成"用户数据"分类，导致临时文件、浏览器缓存混在一起。现在保留原始分类。

**清理虚报。** 之前不管文件有没有删掉都算"清理成功"。现在会实际检测释放了多少空间，没删掉的不算数。

**只读文件处理。** 清理时先去掉只读属性，遇到被占用的文件标记为重启时删除。

## 🗑️ 功能调整

桌面、文档、下载文件夹从迁移列表中移除。这些目录迁移风险太高，出问题整个桌面都打不开，不值得冒这个险。临时文件、系统日志也一样——它们应该被清理而不是被迁移。
